
# House Price Prediction using Machine Learning

## Project Overview
This project predicts house prices using machine learning models.

## Dataset
Housing Prices Dataset from Kaggle.

## Models Used
- Linear Regression
- Random Forest Regressor

## Libraries
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

## Results
Random Forest Regressor performed better than Linear Regression.
